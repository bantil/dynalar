Sora V2 Battery Meter

Source Code
Completed
  • This application was meant for personal use, but anyone can use it. This C# application shows an icon in the system tray for Windows 11 that displays the battery life of the Ninjutso Sora V2 mouse. I had to use Free USB Analyzer to look at the packets being sent to and from the device and my host, and from there add some code that will send the proper feature requests to the HID device to get the required battery information. You will require a Sora V2 mouse as that is the only hardware currently supported. Screenshots are in the readme.
